### 👤 About Me

</div>

I'm a **FullStack Developer** from Brazil 🇧🇷, passionate about building software that is robust, clean, and purposeful. I graduated in **Systems Analysis and Development (ADS)** at IFPR Campus Irati and I'm constantly expanding my expertise.

My journey covers the **full cycle of software development**: from backend APIs with Java and Spring Boot, to mobile interfaces with Flutter, passing through web front-ends and relational databases. I believe great software is built with discipline, deep understanding of architecture, and genuine care for the end user.

- 🔭 Always improving: **Java**, **Spring Boot** & **Flutter**
- 💬 Ask me about **REST APIs**, **Software Architecture** & **Design Patterns**
