Soy un **Desarrollador FullStack** brasileño 🇧🇷, apasionado por crear software robusto, limpio y con propósito. Egresado de **Análisis y Desarrollo de Sistemas (ADS)** en el IFPR Campus Irati, me encuentro en constante evolución técnica y profesional.

Mi recorrido abarca el **ciclo completo del desarrollo de software**: desde APIs back-end con Java y Spring Boot, hasta interfaces móviles con Flutter, pasando por front-ends web y bases de datos relacionales. Creo que el gran software se construye con disciplina, arquitectura sólida y genuino cuidado por el usuario final.

- 🎓 **Egresado en ADS** — IFPR Campus Irati
- 🌱 Profundizando en: **NestJS**, **Prisma** & **Clean Architecture**
- 🔭 Siempre mejorando: **Java**, **Spring Boot** & **Flutter**
- 💬 Pregúntame sobre **REST APIs**, **Arquitectura de Software** & **Design Patterns**
- ✝️ *"Todo lo que hagan, háganlo de corazón, como para el Señor."* — Col. 3:23
